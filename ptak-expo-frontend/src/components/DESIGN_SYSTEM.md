# Design System

Kolory i style z projektu Figma/Locofy.

## Kolory

```scss
--color-dodgerblue: #6f87f6;      
--color-mediumslateblue: #5041d0;  
--color-darkslategray: #2e2e38;   
--color-darkgray: #a7a7a7;        
--color-gainsboro-200: #d7d9dd;   
--color-whitesmoke: #f5f6f7;      
--color-firebrick: #c7353c;       
```

## Typografia

- Roc Grotesk - nagłówki i labele
- Open Sans - tekst body i inputy
- Rozmiary: 11px, 12px, 16px, 18px

## Border radius

- 5px - małe elementy
- 10px - buttony, inputy
- 20px - modale

## Komponenty

Modal: border-radius 20px, tło #f5f6f7
Buttony: primary blue, hover darker blue
Inputy: white background, rounded

Responsive: 2 kolumny desktop, 1 mobile

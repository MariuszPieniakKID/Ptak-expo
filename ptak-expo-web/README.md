# PTAK EXPO Web

Portal dla wystawców PTAK EXPO.

## Instalacja

```bash
npm install
npm start
```

Aplikacja dostępna na http://localhost:3000

## Konfiguracja

Utwórz `.env`:

```env
REACT_APP_API_URL=http://localhost:3001/api/v1
```

- Dashboard wystawcy
